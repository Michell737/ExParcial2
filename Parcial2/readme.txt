Para instalar la aplicación:
- Se instala la librería de manera local: npm install --save ejs

Para configurar la base de datos:
Se tiene que importar en MARIADB (phpmyadmin) el archivo anexo "exparcial.sql"

Para ejecutar la aplicación: 
- ejecutar en el directory principal node app.js
- Se debe acceder desde al "localhost:3000/zombies"
- Para registrar un nuevo zombie, solo basta con darle click al botón de "registrar nuevo zombie"
